
# Neo4j Streaming Graph Example

Graph model simulating a streaming platform like Netflix.

## Nodes
User, Movie, Series, Actor, Director, Genre

## Relationships
WATCHED (rating)
ACTED_IN
DIRECTED
IN_GENRE

## Steps to run

1. Run cypher/create_constraints.cypher
2. Run cypher/load_nodes.cypher
3. Run cypher/load_relationships.cypher
4. Run cypher/business_queries.cypher
